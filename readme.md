# Sistem Penjualan PHP MySQL

Aplikasi sistem penjualan berbasis web menggunakan **PHP & MySQL**.

## Fitur
- Login menggunakan database
- Dashboard dengan sidebar
- CRUD Produk
- CRUD Customer
- Transaksi Penjualan (Kasir)
- Laporan Penjualan Harian & Bulanan

## Teknologi
- PHP Native
- MySQL
- HTML & CSS
- XAMPP
